## 内部标签

```
   <script>
        alert("hello, world");
    </script>
```

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HelloWorld</title>
    <script>
        alert("hello, world");
    </script>
</head>
<body>
    
</body>
</html>
```



## 外部引入

```
<script src="hello.js"></script>
```

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HelloWorld</title>
    <script src="hello.js"></script>
</head>
<body>
    
</body>
</html>
```

```

<script src="hello.js" type="text/javascript"></script>

```
